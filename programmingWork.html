This is technically creative writing

Online Image Editor

An online image editor written in JavaScript which allows a user to upload an image, apply filters, and save the resulting image onto their device

Ideal and Non-Ideal Gas Simulation

A simulation of ideal and non-ideal gasses in 2-dimensions as circles with elastic collisions. For collision detection, I made use of a spatial hash map, which allows the program to handle upwards of 2500 particles without any significant performance drop.

Random Variable Distributions in C

A C library for generating Random Variables according to a few common probability distributions.

ASCII Art Generator

A GUI and terminal program which can create an ASCII artwork from an image. The GUI is made using the tkinter module.

Gravity Simulation

A simple Newtonian gravity simulation written in Python which uses point masses to approximate different gravitational interactions. There are a few prebuilt systems in the code, such as our solar system and a binary star system with circumbinary planets.

Pathfinder

An app written in Java which allows the user to place a start point, end point and obstacles on a grid. The shortest path between the two points around any obstacles is then found and the process is visualized.

Sudoku Game

A sudoku game written in Java using the LibGDX framework. The game doesn't have preloaded puzzles, but rather generates a random new puzzle depending on the selected difficulty. This is done with recursive backtracking.

Wordle Clone

A clone of Wordle, written in Python using the pygame module.

Discord Bot

A music bot I wrote for a personal Discord server. You enter a song you want and the bot will join your channel and play the audio from the most related YouTube video. Songs can be queued, skipped, paused, resumed and stopped entirely. There are a few other fun commands attatched to the bot as well. Sadly, due to some of Replit's software migrations, the bot is no longer running.

Youtube Downloader

A command line app written in Python. You enter the url of a video and it downloads it locally.
