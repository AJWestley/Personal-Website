Gravity Simulation

A simple Newtonian gravity simulation written in Python which uses point masses to approximate different gravitational interactions.

Pathfinder

An app written in Java which allows the user to place a start point, end point and obstacles on a grid. The shortest path between the two points around any obstacles is then found and the process is shown visually to the user.

Sudoku Game

A sudoku game written in Java using the LibGDX framework. The game doesn't have preloaded puzzles, but rather generates a random new puzzle depending on the selected difficulty.

Wordle Clone

A clone of Wordle, written in Python using the pygame module.

Discord Bot

A music bot I wrote for a personal Discord server. You enter a song you want and the bot will join your channel and play the audio from the most related YouTube video. Songs can be queued, skipped, paused, resumed and stopped entirely. There are a few other fun commands attatched to the bot as well.

Youtube Downloader

A command line app written in Python. You enter the url of a video and it downloads it locally.
